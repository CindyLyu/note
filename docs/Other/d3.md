# D3.js

- 全名為（Data-Driven Documents）
- ![d3-function](./img/d3-function.png)
- `scaleLinear` / `scaleLog` 線性/指數縮放
  - domain 輸入數值範圍
  - range 輸出範圍
- `scaleLinear()` 用線性的方式顯示；`scaleOrdinal()` 會是以序列來顯示
- `d3.range(10)` 可以產生 0~9 的內容陣列
