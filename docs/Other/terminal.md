# Terminal

- Linux 預設的 Shell 為 Bash（/bin/bash)
- `source filename`是 Bash Shell 內建的指令，會去執行這個檔案的內容，也可以用 `. filename` 取代
- `npm repo <project>`：打開此 project 的 GitHub
- `npm docs <project>`：打開此 project 的官方文件 <sup>[3]</sup>
